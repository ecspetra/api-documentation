Документация API сервиса LiveLib

Основная информация

Версия протокола v2
Ключ партнера xxxxxxxxxxxxxxxxxxxxxxx
Формат обмена JSON
Кодировка ответа UTF-8
Версия документа 15 февраля 2021

Структура запроса

api_key ключ партнера, можно указывать здесь или передавать GET параметром
query тело запроса
filter фильтр запроса
term параметр фильтра запроса — строка, число или массив
fields поля в ответе — массив, по умолчанию все
sort поле из ответа, по которому сортируются результаты
from первая позиция слайса результатов, по умолчанию 0
size количество результатов в слайсе, по умолчанию 100

Структура ответа

status okay при успешном выполнении
took время выполнения запроса в мс
hits результаты запроса
total количество всех результатов запроса
hits результаты — массив

Информация о книгах

https://api.livelib.ru/v2/bookinfo

Параметры запроса

Параметр Тип Описание
id int array Идентификаторы книг из источника
isbn string array ISBN изданий, может использоваться дополнительно к id или вместо
source string Алиас источника (mybook)

Доступные поля в ответе

Поле Тип Размер Описание
id int 4 Внутренний идентификатор издания на Лайвлибе
source_id int 4 Идентификатор книги в источнике
book_id int 4 Внутренний идентификатор книги на Лайвлибе
name string 255 Название издания
rating int 4 Рейтинг книги в списке Топ-100
author_id int 4 Внутренний идентификатор автора
author string 255 Имя и фамилия автора
isbn string 255 Все isbn издания, перечислены через запятую
year int 4 Год выпуска издания
publishing string 255 Названия всех издательств, выпускавших издание, перечислены через запятую
series string 255 Название издательской серии
description string 64K Аннотация к изданию
count_plus int 4 Количество пользователей, поставивших положительную оценку книге
count_zero int 4 Количество пользователей, поставивших нейтральную оценку книге
count_minus int 4 Количество пользователей, поставивших отрицательную оценку книге
count_readers int 4 Количество пользователей, прочитавших книгу
count_readers_all int 4 Количество прочитавших и недочитавших книгу
tags string 127 Теги к книге, перечислены через запятую
pic_url string 127 Ссылка на обложку издания, png-изображение с шириной 100px
avg_mark float 4 Средняя оценка книги по 5-балльной шкале, формат 4.123
count_reviews int 4 Количество рецензий на книгу
count_quotes int 4 Количество цитат из книги
count_stories int 4 Количество историй о книге
count_selections int 4 Количество подборок, в которые добавлена книга
facts string 16M Текст раздела «Интересные факты» информации о книге
facts_marked string 16M Текст раздела «Интересные факты» информации о книге c html тегами
facts_date string 50 Дата изменения раздела, формат "2021-05-10T11:25"
movies string 16M Текст раздела «Экранизации» информации о книге
widget_url string 127 Ссылка на виджет рейтинга книги
widget_url2 string 127 Ссылка на виджет рейтинга книги в две строки
0_mark int 4 Количество пользователей, поставивших оценку 0 книге
1_mark int 4 Количество пользователей, поставивших оценку 1 книге (по 5-балльной шкале)
2_mark int 4 Количество пользователей, поставивших оценку 2 книге (по 5-балльной шкале)
3_mark int 4 Количество пользователей, поставивших оценку 3 книге (по 5-балльной шкале)
4_mark int 4 Количество пользователей, поставивших оценку 4 книге (по 5-балльной шкале)
5_mark int 4 Количество пользователей, поставивших оценку 5 книге (по 5-балльной шкале)

Пример запроса для получения полной информации по нескольким ID книг

  
  POST https://api.livelib.ru/v2/bookinfo?api_key=2090a7e895d9576ce5715c0b87328d4f
  {
    "query" : {
      "filter": [
        {"term": {"source": "mybook"}},
        {"term": {"id": [280170,22725,60539,299238,292254,206784,295790]}}
      ]
    }
  }

Пример запроса для получения полной информации по нескольким ID и ISBN

  
  POST https://api.livelib.ru/v2/bookinfo?api_key=2090a7e895d9576ce5715c0b87328d4f
  {
    "query" : {
      "filter": [
        {"term": {"source": "mybook"}},
        {"term": {"id": [280170,22725,60539,299238,292254,206784,295790]}},
        {"term": {"isbn": ["5-17-021722-6", "9788280870285"]}}
      ]
    }
  }

Пример запроса для получения источника, названия и ссылки на виджет для первых 3 книг результа при запросе по нескольким ID книг

  
  POST https://api.livelib.ru/v2/bookinfo?api_key=2090a7e895d9576ce5715c0b87328d4f
  {
    "query" : {
      "filter": [
        {"term": {"source": "mybook"}},
        {"term": {"id": [280170,22725,60539,299238,292254,206784,295790]}}
      ]
    },
    "fields": ["id", "source_id", "name", "widget_url"],
    "from": 0,
    "size": 3
  }

Пример запроса для получения “Интересные факты” только для книг, у которых факты обновились после 1 января 2021

  
  POST https://api.livelib.ru/v2/bookinfo?api_key=2090a7e895d9576ce5715c0b87328d4f
  {
    "query" : {
      "filter": [
        {"term": {"source": "mybook"}},
        {"term": {"id": [148706,458089,314194,293066,184358,621234]}},
        {"range": {"facts_date": {"gte": "2021-01-01T00:00"}}}
      ]
    },
    "fields": ["id","source_id","facts","facts_marked","facts_date"]
  }

Информация о рецензиях

https://api.livelib.ru/v2/reviews

Параметры запроса

Параметр Тип Описание
id int array Идентификаторы книг из источника
source string Алиас источника (mybook)
strict bool true - выбрать только рецензии на издания, связанные с источником. По умолчанию false - выбрать рецензии на любые издания книг, связанных с источником

Доступные поля в ответе

Поле Тип Размер Описание
id int 4 Идентификатор рецензии
source_id int 4 Идентификатор книги в источнике
title string 255 Заголовок рецензии
text string 64K Полный текст рецензии
marked_text string 64K Полный текст рецензии с html тегами
reader string 50 Логин автора рецензии
book_id int 4 Идентификатор издания, на которое написана рецензия
review_rating int 4 Рейтинг рецензии
count_comments int 4 Количество комментариев к рецензии
pic_url string 255 Ссылка на изображение профиля автора рецензии, jpg с шириной 100px
date string 50 Дата написания рецензии, формат "2021-05-10T11:25"

Доступные поля для сортировки

review_rating Рейтинг рецензии
date Дата написания рецензии

Пример запроса для получения лучших рецензий на указанные книги с 1 января 2021

  
  POST https://api.livelib.ru/v2/reviews?api_key=2090a7e895d9576ce5715c0b87328d4f
  {
    "query" : {
      "filter": [
        {"term": {"source": "mybook"}},
        {"term": {"id": [148706,458089,314194,293066,184358,621234]}},
        {"range": {"date": {"gte": "2021-01-01T00:00"}}}
      ]
    },
    "fields": ["id","source_id","review_rating","reader","date","marked_text"],
    "sort": {"review_rating":"desc"}
  }

Информация об авторах

https://api.livelib.ru/v2/authors

Параметры запроса

Параметр Тип Описание
id int array Внутренний идентификатор автора на Лайвлибе

Доступные поля в ответе

Параметр Тип Размер Описание
id int 4 Внутренний идентификатор автора
name string 255 Имя и фамилия автора
born_date string 50 Дата рождения, формат "2014-10-03"
died_date string 50 Дата смерти, формат "2014-10-03"
pic_url string 127 Ссылка на фотографию автора, jpeg-изображение максимального доступного размера
has_picture int 1 Есть ли фотография у автора
biography string 16M Текст раздела «Биография» информации об авторе
facts string 16M Текст раздела «Интересные факты» информации об авторе
links string 16M Текст раздела «Ссылки» информации об авторе

Пример запроса для получения ссылок на фото авторов

  
  POST https://api.livelib.ru/v2/authors?api_key=2090a7e895d9576ce5715c0b87328d4f
  {
    "query" : {
      "filter": [
        {"term": {"id": [1,31,48,75]}},
        {"term": {"has_picture": [1]}}
      ]
    },
    "fields":["id","pic_url"]
  }

Пример кода PHP для запроса к API

            
  
  <?php
  
  $source = 'mybook';
  $ids = [280170,22725,60539,299238,292254,206784,295790];
  
  $json_data = '
    {
      "query" : {
        "filter": [
          {"term": {"source": "mybook"}},
          {"term": {"id": [' . implode(',', $ids) . ']}}
        ]
      },
      "fields": ["id", "source_id", "name", "widget_url"],
      "from": 0,
      "size": 3
    }
  ';
  
  $api_key = '2090a7e895d9576ce5715c0b87328d4f';
  $url = 'https://api.livelib.ru/v2/bookinfo?api_key=' . $api_key;
  
  
  $ch = curl_init();
  
  curl_setopt($ch, CURLOPT_URL, $url);
  curl_setopt($ch, CURLOPT_POST, 1);
  curl_setopt($ch, CURLOPT_POSTFIELDS, $json_data);
  curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
  
  $server_output = curl_exec ($ch);
  print_r(curl_error($ch));
  curl_close ($ch);
  
  print_r($server_output);

Ошибки

Если при выполнении запроса произошла ошибка, то придет ответ в формате

  
  {
    "status": "error",
    "error_code": "11xx",
    "message": "error description"
  }
Код ошибки Описание
1101 Не указан api_key
1102 api_key недействителен
1103 Пустой POST-запрос
1107 В запросе не указаны id книг / объектов
1108 Не указан параметр source
1109 Источник не найден или недоступен для переданного api_key
